Project Overview

The Outreach project serves the community by exposing the youth of Columbus to computer science through workshops. This is foundational to promoting diversity in computing professions. Outreach meets weekly to build modules which teach basic computer programming. These modules can be games, activities, or anything else you can think of! Once completed, we present these modules in workshops at local schools.


Current Modules

  • World's Hardest Game
    Students must modify the code to beat Level 3. They can also change the images used for the player, the obstacles, the goal, and the background.
  • Brutus Ball
    Students write code to compete against a computer player or each other! The last Brutus Buckeye standing wins.

Upcoming Modules

  • Cats and Mice
    In this platformer, you're a mouse. Your goal is to get the cheese while avoiding the cats.
